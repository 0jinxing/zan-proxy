简介

ZanProxy

ZanProxy是一个用Node.js编写的HTTP代理服务器，可用于修改请求地址和模拟响应数据。它同时也是一个自定义DNS解析和请求监控的工具。 该代理服务器有人性化的界面，简单易用。我们还为开发者提供了插件机制来自定义代理行为。

特点

  1. 支持HTTP，HTTPS和Websocket
  2. 支持修改请求地址
  3. 支持模拟响应数据
  4. 支持自定义DNS解析
  5. 支持自定义插件
  6. 图形化配置
